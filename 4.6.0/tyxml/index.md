
# tyxml index


## Library tyxml

This library exposes the following toplevel modules:

[`Tyxml`](./Tyxml.md) Typed implementation for HTML, SVG and XML
[`Tyxml_html`](./Tyxml_html.md) Typesafe constructors and printers for Html documents.
[`Tyxml_svg`](./Tyxml_svg.md) Typesafe constructors and printers for Svg documents.
[`Tyxml_xml`](./Tyxml_xml.md) Basic functions for construction and manipulation of XML tree.

## Library tyxml.functor

This library exposes the following toplevel modules:

[`Html_f`](./Html_f.md) Typesafe constructors for HTML documents (Functorial interface)
[`Html_sigs`](./Html_sigs.md) HTML signatures for the functorial interface.
[`Html_types`](./Html_types.md) HTML types with variants, goes with Html\_sigs.T.
[`Svg_f`](./Svg_f.md) Typesafe constructors for SVG documents (Functorial interface)
[`Svg_sigs`](./Svg_sigs.md) SVG signatures for the functorial interface.
[`Svg_types`](./Svg_types.md) SVG types with variants, goes with Svg\_sigs.T.
[`Xml_iter`](./Xml_iter.md) Basic iterators over XML tree (functorial interface).
[`Xml_print`](./Xml_print.md) Printing utilities.
[`Xml_sigs`](./Xml_sigs.md) XML Signatures.
[`Xml_stream`](./Xml_stream.md) Streaming IO to/from XML trees
[`Xml_wrap`](./Xml_wrap.md)
